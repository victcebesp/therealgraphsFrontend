Data source

The data used to build the visualization is obtained from an open source GitHub repository operated by the Johns Hopkins University Center for Systems Science and Engineering (JHU CSSE) and supported Supported by ESRI Living Atlas Team and the Johns Hopkins University Applied Physics Lab (JHU APL).

This raw data has been preprocessed as shown in the tutorial and it is daily updated using AWS.
