Graphs construction tutorial

In this section it will be explained how both graphs showing in the dashboard are built to be fully transparent.

Bar chart

In the bar chart, each bar is representing the percentages of active, fatal and recovered cases in a country at a specific date selected using the slider. It is important to note that only the top fifteen more fatally-affected countries are shown. Additionally, China's bar is displayed separated from the rest of the bars to make it easier to compare it with the rest of the countries.

Active cases' percentage

In order to calculate active cases from the reported confirmed cases from the data source, recovered and fatal cases are discounted from confirmed cases. To calculate this, the following equation is used: \[active = {confirmed - recovered - fatal}\]

Finally, these newly-calculated active cases are divided over the total confirmed cases as follows: \[percentage = {active \over confirmed}\]

Fatal cases' percentage

The calculation of the fatal cases' percentage is simpler. The number of fatal cases each day is divided by the total amount of confirmed cases following the equation: \[percentage = {fatal \over confirmed}\]

Recovered cases' percentage

The calculation of the recovered cases' percentage is very similar to the fatal cases one. The number of recovered cases each day is divided by the total amount of confirmed cases following the equation: \[percentage = {recovered \over confirmed}\]

Line chart

In the line chart, each line represents the daily evolution of active cases in a country until a certain date selected by the slider. To know which specific country each line is representing, it has been added a legend on the right-hand side. It is important to note that the same top fifteen more fatally-affected countries are shown. In this case, China's data is not plotted separately as it is interesting comparing it in the same axes as the rest of countries.
